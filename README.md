# Website Profil Kelompok - Tugas Besar Praktikum Version Control

Project ini dibuat untuk memenuhi Tugas Besar mata kuliah Praktikum Version Control. Fokus utama dari tugas ini adalah implementasi alur kerja Git & GitHub secara kolaboratif.

## Deskripsi Tugas
Setiap anggota kelompok berkontribusi dalam repository ini dengan alur kerja standard:
1. Clone repository.
2. Buat branch baru untuk fitur masing-masing (`kontribusi-[NIM]`).
3. Commit perubahan secara berkala.
4. Lakukan Pull Request ke branch utama.

## Cara Berkontribusi
1.  **Clone repository** ini ke komputer lokal Anda.
    ```bash
    git clone https://github.com/JafarSiddik/TugasBesar-PraktikumGit-Kelompok.git
    ```
2.  **Buat branch baru** dengan format `kontribusi-[NIM]`.
    ```bash
    git checkout -b kontribusi-12345678
    ```
3.  **Lakukan perubahan** pada file yang menjadi tanggung jawab Anda (misal: `member1.html`).
4.  **Commit perubahan** dengan pesan yang jelas.
    ```bash
    git add .
    git commit -m "Menambahkan biodata pada member1.html"
    ```
5.  **Push branch** ke GitHub.
    ```bash
    git push origin kontribusi-12345678
    ```
6.  **Buat Pull Request** di GitHub dan minta review dari ketua kelompok.
